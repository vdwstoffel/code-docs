# INSERT

## Create a new record in a table.

```sql
INSERT INTO table_name (column1, column2)
VALUES ('value1', 'value2');
```

## Insert multiple records

Insert multiple records into a table.

```sql
INSERT INTO table_name (column1, column2)
VALUES ('value1', 'value2'),
       ('value3', 'value4');
```
