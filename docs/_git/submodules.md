# Submodules

## Add submodule

```bash
git submodule add <repo_url> <path>
```

## Initialize submodule

```bash
git submodule init
```

## Update submodule

```bash
git submodule update --remote --merge
```

## Remove submodule

```bash
git rm <path>
```
