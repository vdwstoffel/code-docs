# HTML/CSS/JS

## Centering an element horizontal/vertical

<Tabs>
<TabItem value='Html'>

```html
<body>
  <div class="outerContainer">
      <div class="container">Div place nicely in the center</div>
  <div>
</body>
```

</TabItem >
<TabItem value='CSS'>

```css
.outerContainer {
  position: relative;
}

.container {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
  padding: 5px;
}
```

</TabItem >
</Tabs>

<body
  style={{
    border: "1px solid black",
    padding: "5px",
    height: "250px",
    width: "450px",
    position: "relative",
  }}
>
  <div
    style={{
      position: "absolute",
      top: "50%",
      left: "50%",
      transform: "translate(-50%, -50%)",
    }}
  >
    Div place nicely in the center
  </div>
</body>

## Creating a loading spinner

<Tabs>
<TabItem value='Html'>

```html
<div class="loader"></div>
```

</TabItem >
<TabItem value='CSS'>

```css
.loader {
  border: 16px solid #f3f3f3;
  border-top: 16px solid #3498db;
  border-radius: 50%;
  width: 120px;
  height: 120px;
  animation: spin 2s linear infinite;
}

@keyframes spin {
  0% {
    transform: rotate(0deg);
  }
  100% {
    transform: rotate(360deg);
  }
}
```

</TabItem >
</Tabs>
